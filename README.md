## ✦ Features

| # | Feature | Description |
|---|---------|-------------|
| 01 | **Pomodoro Timer** | Structured focus sprints with custom intervals |
| 02 | **Priority Tasks** | Three tasks per day — the ones that actually matter |
| 03 | **Habit Grid** | Minimal streak tracker for daily consistency |
| 04 | **Daily Review** | 60-second end-of-day reflection |
| 05 | **Focus Mode** | One task, full screen, zero distraction |
| 06 | **Offline First** | No account needed — data stays on your device |

## 🚀 Deploying the Website

This site is hosted via **GitHub Pages**.

1. Push all files to the `main` branch
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Save — your site will be live in minutes
